# Content Attribution & Disclaimer

## Content Attribution

This project is an independent educational web application created to help IB Biology students organize their revision through interactive flashcards, note-taking, and active recall.

The flashcard bullet points and syllabus structure are adapted from **Biology for Life** and are **used with the express permission of the website's creator**. 
I am grateful for their permission to incorporate this material into this project.

Biology for Life: https://www.biologyforlife.com/

## Purpose of This Project

The primary goal of this project is to demonstrate full-stack software development skills through the creation of an educational web application. 
Features include user authentication, persistent user progress, personalized notes, review sessions, 
and an interactive flashcard interface built with Vue.js, Flask, and PostgreSQL.

This application is intended to be used as a study companion that encourages active recall and helps students organize their IB Biology revision.

## Answer Explanations

The answer explanations included in the flashcards are designed as concise review references rather than comprehensive textbook-style explanations. 
AI tools were used to assist in generating initial answer drafts, which were reviewed and edited for educational use.

For detailed explanations and additional study materials, users are encouraged to use Biology for Life alongside this application and consult other IB Biology resources.
